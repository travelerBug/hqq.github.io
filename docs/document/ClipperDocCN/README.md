## ClipperLib文档中文版

Clipper库是目前计算机图形届广为使用的图形处理库，可以用于解决平面二维图形的**多边形简化、布尔运算和偏置处理**，在CAD、加工路径与3D打印方面都有着比较重要的应用。本文档基于原英文文档进行翻译，由于译者能力有限，尽量逼近原文档的内容和风格。
本文使用Love2.io驱动；
