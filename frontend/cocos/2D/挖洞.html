# 正多边形顶点位置的计算

  • 公式如下：
for (let i = 0; i < n; i++) {
const r = 2 * Math.PI * i / n;
let vec_x = DIG_RADIUS * Math.cos(r);
let vec_y = DIG_RADIUS * Math.sin(r);
}

理解方式： 一圈360度 正多边形相当于构建多个三角形

已知中间点坐标,求顶点坐标

  • 2 * Math.PI * i / n; 每个顶点的角度(弧度制)
  • 360弧度制 表示为2*Math.PI
